# Genie in a Bottle


## Live Demo: https://genie-in-a-bottle.herokuapp.com/

## Getting Started
- Terminal Commands
 1. Clone repo
 2. npm install
 3. createdb genie-in-a-bottle
 4. npm start
 5. npm run seed
- Enjoy Shopping
